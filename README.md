# Workout Logger API

A RESTful API for tracking workouts, exercises, and fitness progress over time. Log workout sessions, track sets/reps/weight, and monitor your strength gains.

## Features
- Create and manage workout sessions
- Log exercises with sets, reps, and weight
- Track progress and personal records over time
- Filter workouts by date range
- Calculate workout statistics and volume

## Tech Stack
- **Java 21** - Programming language
- **Spring Boot 3.2.x** - REST API framework
- **Spring Data JPA** - Database ORM and repository layer
- **H2 Database** - In-memory database for development
- **Maven** - Dependency management and build tool
- **Lombok** - Reduces boilerplate code

## Getting Started

### Prerequisites
- JDK 21 or higher
- Maven 3.6+

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/workout-logger.git

# Navigate to project directory
cd workout-logger

# Build the project
mvn clean install

# Run the application
mvn spring-boot:run
```

The API will be available at `http://localhost:8080`
